## üõ†Ô∏è Quick Start

### Prerequisites

- Node.js 16+
- npm

### Clone & Go

```shell
git clone https://github.com/asmitsaw/STARTX-Starterpack-of-your-career.git
cd STARTX-Starterpack-of-your-career
npm install
cp sample.env .env   # Configure secrets in .env
```

### Launch Locally

```shell
npm run dev
```

- To set up messaging:  
  Run `setup-messenger.bat` and follow `MESSENGER_SETUP.md`
- Integrate payments:  
  See `STRIPE_SETUP.md` and run tests with `TEST_STRIPE_NOW.md`
- Fix auth issues:  
  Guided steps in `CLERK_AUTH_SETUP.md` & `AUTHENTICATION_FIX.md`
- Explore UI templates:  
  `index.html` (landing), `ai-interview.html`, `profile-excellence.html`
